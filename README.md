
# permApprox Manuscript


This repository contains the code and materials needed to reproduce the simulation studies and real-data applications presented in the manuscript
**“permApprox: a general framework for accurate permutation p-value approximation”**.
All figures, tables, and numerical results reported in the paper can be generated using the scripts provided here.

## Relation to the permApprox R package

The methods proposed in the manuscript are implemented in the open-source R package **`permApprox`**, available at
[https://github.com/stefpeschel/permApprox](https://github.com/stefpeschel/permApprox).

While the package provides a stable, user-facing implementation with default settings, this repository contains extended code used for simulations and methodological evaluations, including additional tuning options and diagnostics that are not exposed in the package interface.

## How to Reproduce the Results

### 1. Clone the repository

``` r
git clone https://github.com/stefpeschel/permApprox-manuscript.git
```

### 2. Open the project in RStudio

Open the `.Rproj` file or the project folder in RStudio.

### 3. Restore the R environment

Use the [`renv`](https://rstudio.github.io/renv/) package to restore the exact package versions:

``` r
renv::restore()
```

### 4. Render the .Rmd files

The file `projthis_code.R` serves as the main entry point for rendering the `.Rmd` files provided in this repository. Individual studies can also be run independently by rendering the corresponding \texttt{.Rmd} files, e.g., 
`rmarkdown::render("simulations/perm_ttest/05_p_approx_methods.Rmd", output_format = "all")`.

## System Requirements

The code was developed and tested using R ≥ 4.5 on Windows and Linux. 
Parallel execution is supported.

## Reproducibility

All simulation scripts set random seeds internally to ensure reproducible results.
Exact numerical values may still vary slightly across platforms due to floating-point
arithmetic and parallel execution.
